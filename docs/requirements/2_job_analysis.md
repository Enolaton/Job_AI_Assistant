# 기능 2: 직무 분석 (Job Analysis) - 요구사항 명세서 초안

## 1. 기능 개요 및 목표 (Overview & Goal)
*   **목표:** 사용자가 채용 공고(JD) URL이나 텍스트를 입력하면, AI가 해당 직무에서 요구하는 핵심 역량을 분석하고 사용자의 '내 경험 뱅크' 데이터와 매칭하여 적합도를 평가해준다.
*   **핵심 가치:** 내가 가진 경험 중 어떤 경험이 해당 직무에 가장 강력한 무기가 될 수 있는지 파악하고, 자기소개서를 쓸 때 어떤 포인트를 강조해야 하는지 인사이트를 제공한다.

## 2. 사용자 흐름 (User Flow)
1.  **채용 공고 입력:** '직무 분석' 페이지 진입 후 채용 공고의 URL 또는 공고 내용을 직접 복사하여 붙여넣기.
2.  **직무 분석 요청:** `분석 시작` 버튼 클릭.
3.  **AI 로딩 대기:** AI가 공고 내용을 읽고 핵심 키워드(Hard Skill, Soft Skill, 우대사항 등)를 분석하는 동안 대기.
4.  **분석 결과 확인 1 (직무 정보):** 해당 직무의 핵심 업무, 필수 역량, 우대 사항 등이 구조화되어 요약된 내용 확인.
5.  **내 경험 매칭 (AI 연동):** '내 경험 뱅크'에 저장된 사용자의 경험 데이터와 방금 분석한 직무 요건을 비교.
6.  **분석 결과 확인 2 (매칭도 & 추천 전략):**
    *   나의 직무 적합도 점수(또는 강점 평가) 확인.
    *   어떤 경험을 메인으로 어필해야 좋을지 추천 전략 확인.
    *   부족한 역량이나 보완해야 할 점에 대한 피드백 확인.
7.  **저장 및 자소서 작성으로 이동:** 이 분석 결과를 저장하고, 바로 '이 공고로 자소서 쓰기' 버튼을 눌러 기능 3으로 이동.

## 3. AI 연동 시나리오 (AI Interactions)

### 3.1. 채용 공고(JD) 분석
*   **Trigger:** 공고 텍스트/URL 입력 후 분석 요청.
*   **Input Data:** JD 원문 텍스트.
*   **System Prompt (예시):**
    > "너는 IT/테크 채용 전문가야. 다음 채용 공고 원문을 읽고, (1) 주요 업무, (2) 필수 하드스킬, (3) 요구되는 소프트스킬, (4) 우대사항을 JSON 형태로 분류하고 요약해줘."
*   **Output Format:** JSON 구조 (주요 업무, 필수 역량, 우대사항 등)

### 3.2. 내 경험 매칭 
*   **Trigger:** 채용 공고 분석 완료 후, 유저의 '내 경험 뱅크' 데이터에 접근.
*   **Input Data:** JD 분석 결과 데이터 + 사용자의 모든 '내 경험 뱅크' 구조화된 텍스트.
*   **System Prompt (예시):**
    > "너는 인사담당자야. 채용 공고 요건과 지원자의 경험 데이터를 비교해서, 이 공고에 지원할 때 가장 강력하게 어필할 수 있는 경험 3가지를 매칭하고 그 이유를 설명해줘. 지원자의 강점과 보완점도 함께 제시해."
*   **Output Format:** JSON (매칭 점수 또는 평가 등급, 추천 경험 ID 리스트, 강점, 보완해야 할 점)

## 4. 데이터 모델 (Data Model - DB 스키마 초안)
*   `JobAnalysis` (직무 분석 테이블)
    *   `id` (PK)
    *   `userId` (FK, 작성자)
    *   `companyName` (String, 예: "네이버")
    *   `jobTitle` (String, 예: "프론트엔드 개발자")
    *   `jdText` (Text, 공고 원문)
    *   `jdAnalysisResult` (JSON, 파싱된 직무 요건)
    *   `matchingResult` (JSON, 매칭도 및 추천 전략)
    *   `createdAt`

## 5. UI/UX 요구사항 (UI/UX Requirements)
*   **입력 폼:** JD 텍스트 입력창은 길게 입력할 수 있도록 크게 배치하고, URL 입력 방식도 지원하면 사용자 편의성 향상.
*   **시각화:** 직무 적합도나 역량 매칭 결과를 단순 텍스트가 아닌 '방사형 차트'나 '게이지 바' 등 시각적 요소로 표현하여 직관성을 높임.
*   **경험 카드 연동:** 추천된 내 경험 카드가 결과 화면에 바로 노출되도록 디자인하여, 분석된 결과와 경험의 연결 고리를 강조.
