# 🚀 AI-Powered JD Optimizer & Resume Builder
> **구직자의 UX Friction을 최소화하는 AI 기반 맞춤형 JD 추천 및 자소서 최적화 솔루션**

본 프로젝트는 채용 시장 내 구직자가 직면하는 정보 비대칭성을 해소하고, 공고 분석부터 최종 지원까지의 **사용자 경험 마찰(UX Friction)**을 최소화하는 것을 목표로 하는 AI 기반 채용 최적화 솔루션입니다.

---

## 📌 1. 프로젝트 개요 및 전략 (Project Overview)
기존의 단순 키워드 매칭 방식이 해결하지 못한 **인지 부하(Cognitive Load)** 문제를 해결합니다. JD URL 입력 즉시 수행되는 **AI-driven Heuristic Evaluation**을 통해 핵심 요구사항을 정교하게 추출하고, 사용자의 '경험 저장소'와 결합하여 서류 합격률을 극대화합니다.

### 🎯 Key Strategy
* **JD 분석 자동화:** URL 입력만으로 기술 스택(Hard Skills)과 조직 문화(Soft Skills) 정밀 분석.
* **심리스 워크플로우:** 기업 분석 ➔ 자소서 초안 ➔ 면접 준비로 이어지는 통합 환경 제공.
* **데이터 기반 피드백:** 인사담당자와 실무자라는 두 가지 관점의 교차 검증 피드백 제공.

---

## ⚙️ 2. 서비스 아키텍처 및 유저 플로우
시스템은 사용자의 데이터 보유 상태에 따라 유연하게 대응하며, 모든 경로는 **JD 분석 엔진**을 관통합니다.

### 🔄 Data Path Visualization
`[JD URL Input]` ➔ `[NLP Analysis Engine]` ➔ **[3-Way Strategic Output]**
1. **Output A (자소서 생성/평가):** 직무 역량 기반 가이드라인 도출.
2. **Output B (기업 분석 리포트):** 재무 및 사업 전략 데이터 시각화.
3. **Output C (면접 준비):** JD 및 작성된 자소서 기반 동적 질문 생성.

### 🛤️ 사용자 상태별 분기 로직
* **기보유 사용자:** `경험 선택` ➔ `JD 역량 매칭` ➔ `초안 생성`
* **미보유 사용자:** `경험 입력 가이드 진입` ➔ `AI 초안 가이드 제공` ➔ `유사 직무 경험 도출`

## 🎨 3. UI/UX 디자인 가이드 (Interface)
AI 협업 효율을 극대화하기 위해 **'편집 공간(User)'**과 **'참조 공간(AI Assist)'**을 전략적으로 분리(Split View)하였습니다.

### 📑 자소서 작성 및 AI 협업 센터
| 구분 | 주요 구성 요소 및 기능 | Expected User Interaction |
| :--- | :--- | :--- |
| **상단 영역** | 문항 탭 (1~4번 공통 + 추가) | Tab Click: 문항 간 컨텐츠 즉시 스위칭 |
| **좌측 영역** | 사용자 직접 입력 편집기 | Direct Type: 실시간 저장 및 자유 편집 |
| **우측 영역** | AI 어시스턴트 카드 | Click: [초안 생성], [평가], [소제목 추천] |
| **플로팅** | 기업 분석 리포트 버튼 | Pop-up: 작업 중단 없이 기업 정보 상시 참조 |

## 🧠 4. 핵심 기능 모듈 상세
### ✅ AI-driven JD 분석 엔진
단순 텍스트 파싱을 넘어, JD 내 기술된 핵심 요구 역량과 조직 문화를 추출하여 서비스 전반의 준거 집단으로 활용합니다.

### ✅ 멀티 관점 피드백 엔진 (Dual Feedback)
* **실무담당자 관점(Practitioner):** Hard Skills 및 Tech Stack 매칭률 중심 평가.
* **인사담당자 관점(HR):** 인재상 및 핵심 가치 기반의 조직 적합도(Cultural Fit) 평가.

### ✅ Contextual 기업 분석 리포트 (모달 형태)
* **기본 정보:** 설립일, 기업 유형, 업종, 대표자 등.
* **가치 체계:** 인재상, 비전, 미션 스테이트먼트.
* **재무 데이터:** 사원수 추이, 매출액, 영업이익, 자본금 규모.
* **전략/트렌드:** 사업 구조 분석, 리스크 요약, 최근 3개월 주요 뉴스 요약.

---

## 🛠️ 5. 시스템 구축 요건 (Scope)
* **메인/홈페이지:** JD URL 입력 인터페이스 및 대시보드.
* **Authentication:** 세션 유지(기억하기) 기능이 포함된 보안 모듈.
* **AI 협업 센터:** 좌/우 분할 레이아웃 및 실시간 AI 통신(High Complexity).
* **마이페이지:** Side Bar 기반의 [경험 저장소], [작성 목록], [보관함] 관리.
* **면접 준비 섹션:** 자소서 컨텐츠와 JD 역량을 Cross-referencing한 질문 생성기.

---

## 📈 데이터 연동 구조 (System Data Flow)
1. **Source:** JD URL로부터 기업 재무 및 직무 요건 데이터 크롤링/추출.
2. **Process:** 추출 데이터를 '기업 분석 리포트'로 전달함과 동시에 피드백 엔진의 '평가 기준값'으로 설정.
3. **Integration:** 사용자 경험 저장소 데이터와 결합하여 맞춤형 초안 및 면접 질문으로 최종 가공.
